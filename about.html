ReScience Lab is addressing two critical challenges in computational science:

  1. Reproducibility Validation for computational scientific experiments
  2. Coding Ability Enhancement for interdisciplinary researchers without specialized programming backgrounds

Our mission is to standardize computational science experiments through the CRESP protocol (Computational Research Environment Standardization Protocol), making scientific research more accessible, reliable, and verifiable.

CRESP Documentation | CRESP Documentation

CRESP - A standardized protocol for computational research environment description and reproduction

https://cresp.resciencelab.ai/

CRESP is our answer to the inconsistency in computational research environments. This open-source protocol:

  • Standardizes computational science experiment formats
  • Provides a framework for reproducibility validation
  • Creates a distributed system for verification of scientific results
  • Supports researchers across disciplines regardless of coding proficiency

Also this project is going to be shared on PyCon UK 2025.

E2M - Everything to Markdown (1.1k+ Stars ⭐)

E2M is a Python library that converts various file types into Markdown format using a parser-converter architecture. It supports multiple file formats including doc, docx, epub, html, pdf, ppt, pptx, mp3, and m4a. The project's ultimate goal is to provide high-quality data for Retrieval-Augmented Generation (RAG) and model training or fine-tuning.

This open source project has gained over 1,100 stars on GitHub and serves as a comprehensive solution for document conversion needs in AI and data processing workflows.
